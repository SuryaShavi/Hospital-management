# Hospital Management System – Figma AI Prompt (React + Tailwind + Vite)

## Project Goal

Design a **modern, professional, and user-friendly Hospital Management System web application UI**.

The system should allow hospitals to:

* Manage patients
* Store medical records
* Schedule appointments
* Manage doctors and staff
* Handle billing
* Upload medical documents
* Track treatments and reports

The design should look **clean, trustworthy, and professional**, similar to modern healthcare dashboards.

The UI must be structured so developers can easily implement it using:

* **React**
* **Tailwind CSS**
* **Vite**

The project should run using:

npm install
npm run dev
npm start

---
